== Description ==

### W3Scroll Top is created to scroll to top  of the site. User can go to top the site using this tool easily. W3Scroll Top plugin is created with jQuery and <a href="http://w3itinfo.com/plugins/w3scrolltop" target="_blank">Live Preview:</a> Plugin is created by <cite><a href="http://w3itinfo.com/" title="SM Faruque">SM Faruque</a>.</cite>

Plugin Features

* Just Regular Installation in your theme and active plugin.
* Once active W3Scroll Top plugin it will work for your WordPress Theme.
* Very Lightweight. Only 10KB.

== Installation ==

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once active W3Scroll Top plugin, don't need to do anything, it will work for your WordPress Theme.
